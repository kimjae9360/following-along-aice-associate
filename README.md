# 따라하며 끝내는 AICE Associate

> 파이썬을 몰라도 따라만 하면 AICE Associate 합격까지 갈 수 있도록 만든 실전형 학습 노트북 시리즈 + 프리미엄 합격 가이드입니다.

AICE Associate는 "이론 자격증"이 아니라 "실무 도구 사용 자격증"에 가깝습니다. 데이터 로딩 → EDA → 전처리 → 머신러닝/딥러닝 모델링 → 성능 평가로 이어지는, 실제 데이터 분석 업무의 흐름을 손으로 직접 짜보게 만드는 시험입니다. 이 저장소는 그 흐름을 **이론 설명 → 예제 코드 → TODO 실전문제 → 정답(주석 포함)** 구조로 반복 학습할 수 있게 구성했습니다.

## 🚀 시작하는 방법

### 1. 환경 준비

```bash
# Python 3.10~3.12 권장
python -m venv .venv
source .venv/Scripts/activate   # Windows: .venv\Scripts\activate

pip install pandas numpy matplotlib seaborn scikit-learn xgboost imbalanced-learn tensorflow jupyter nbformat
```

### 2. 학습 순서

1. `00_시작하기/00_실력진단테스트.ipynb`로 내 약한 영역부터 확인
2. `00_시작하기/00_시험개요_오픈북전략.ipynb`로 시험 구조·오픈북 전략 파악
3. `01_NumPy_Pandas_기초` → `10_추가연습` 순서대로, 각 폴더의 "이론+실무" 노트북 → "실무전용" 노트북 순으로 반복 학습
4. `09_모의고사` 10회분으로 실전 감각 익히기 (90분 타이머 맞추고 풀기)
5. `99_전체통합본.ipynb`로 마무리 총복습
6. `학습진도체크리스트.md`에 진도와 모의고사 점수를 기록하며 관리

### 3. 막힐 때 참고 자료

- `00_시작하기/00_AICE_에러_해결가이드.ipynb` — 자주 나오는 에러 메시지와 해결 코드
- `00_시작하기/00_AICE_핵심_코드_치트시트.ipynb` — 실전 기출 위주 코드 뼈대 모음
- `00_시작하기/aice_grader.py` — `import aice_grader as grader` 후 `grader.check_nulls(df)`처럼 한 줄로 결측치/인코딩/shape/점수를 자동 점검
