# 10_추가연습 폴더는 무엇인가요?

이 폴더는 **09_모의고사와는 목적이 다릅니다.** 헷갈리기 쉬워서 먼저 차이를 정리합니다.

| 구분 | 09_모의고사 | 10_추가연습 (이 폴더) |
|---|---|---|
| 형식 | 이론 설명 없이 TODO만 (실전 시험처럼) | **이론 설명 + 예제 코드 + TODO** (배우면서 풀기) |
| 목적 | 실전 감각·시간 관리 훈련 | **다른 데이터셋에 응용하며 개념을 다시 확인** |
| 데이터 | 10가지 서로 다른 데이터 (Titanic, 유방암, 손글씨 등) | 회귀·다중분류·불균형 3가지 유형 데이터 |
| 언제 풀면 좋은가 | 01~08 학습을 다 마친 뒤, 시험 직전 | **01~08 각 주제를 배운 직후**, 바로 이어서 |

한 문장으로: **09번은 "다 배운 뒤 실전처럼 풀어보기"**이고, **10번은 "방금 배운 개념을 낯선 데이터에도 적용할 수 있는지 바로 확인하기"**입니다.

## 이 폴더에 있는 파일들

메인 학습 노트북(01~08)은 전부 **Titanic 데이터** 하나로 진행됩니다. 같은 데이터만 계속 쓰면 "이 코드는 Titanic에서만 되는 건가?" 하는 착각이 생기기 쉽습니다. 그래서 이 폴더는 **일부러 전혀 다른 데이터셋**으로 같은 유형의 문제를 다시 연습하도록 만들었습니다.

| 파일 | 데이터 | 유형 | 메인 노트북과의 대응 |
|---|---|---|---|
| EX01_회귀_실전문제_캘리포니아집값.ipynb | 캘리포니아 집값 | 회귀 | 05_머신러닝_회귀, 08_딥러닝(회귀) |
| EX02_다중분류_실전문제_아이리스품종.ipynb | 아이리스 품종 | 다중분류 | 07_머신러닝_분류_다중, 08_딥러닝(다중분류) |
| EX03_불균형데이터_실전문제_고객이탈.ipynb | 통신사 고객 이탈 | 이진분류(불균형) | 06_머신러닝_분류_이진, 08_딥러닝(이진분류) |

각 파일 뒤에 `_실무전용.ipynb`가 붙은 버전은 **이론 설명 없이 TODO 문제만** 모아둔 응용 연습용입니다 (개념을 이미 이해했다는 전제하에, 손으로 더 많이 풀어보고 싶을 때 사용하세요).

## 추천 학습 순서

1. `05_머신러닝_회귀` 폴더를 다 풀었다면 → 바로 `EX01_회귀_실전문제_캘리포니아집값.ipynb`로 이어서 응용
2. `06_머신러닝_분류` 폴더를 다 풀었다면 → 바로 `EX03_불균형데이터_실전문제_고객이탈.ipynb`로 이어서 응용
3. `07_머신러닝_분류_다중`을 다 풀었다면 → 바로 `EX02_다중분류_실전문제_아이리스품종.ipynb`로 이어서 응용
4. 모든 걸 다 마쳤다면 마지막으로 `09_모의고사`에서 실전처럼 최종 점검
